get_ring_elts_direct(init_theta, prec[, eps])

Given an angle θ, find elements u, t ∈ Z[ω], and an integer k such that the matrix (1/√2^k) | u -t^* | | t u^* | approximates a Z-rotation over the angle theta, i.e. the operator | exp(-iθ/2) 0 | | 0 exp(iθ/2) |.

is_reducible(u)

Given a value u ∈ Z[ω], determine that supposing if we had some w = (1/√2^k)u ∈ D[ω], we could re-write w as w = (1/√2^(k-1))u' ∈ D[ω] for some u' ∈ Z[ω].

reduce(u)

Given a u ∈ Z[ω] such that is_reducible(u) is True, return u' such that (1/√2^k)u = (1/√2^(k-1))u'.
